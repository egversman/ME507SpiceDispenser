void task_opening ( void *  p_params)

Task Opening.

Task Opening.

This task creates an opening motor object that cross references its position with a hall effect sensor to open and close. When the hall effect sensor reads a signal, the wheel is in the open position. To close, the wheel rotates for a set duration that was chosen through testing. This did not need to be precise as the vast majority of the wheel represents the "closed" state.
