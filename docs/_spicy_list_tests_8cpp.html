Detailed Description

This file contains 39 test cases for SpicyLinkedList, SpicyNode, and SpicySort. This file is used for testing process and should not be read by a microcontroller. In order to run these test cases, move this file, along with listTestMain.cpp SpicyNode.cpp, SpicyLinkedList.cpp, SpicySort.cpp, and all corresponding header files to a separate directory on your local machine. All instances of uint8_t must be changed to int and uncomment the #includes in SpicyLinkedList.h that are labeled "for testing only." Compile using g++.
