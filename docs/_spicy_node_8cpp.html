Detailed Description

This file contains a linked list node implementation. Each SpicyNode contains the data for one element within a user input request, as well as a pointer to the next node in a SpicyLinkedList. The collection of all user data is stored as several linked SpicyNodes in a SpicyLinkedList.

uint8_t spiceId

Node class to represent an element of a SpicyLinkedList.

While a traditional node in a linked contains one value and a node pointer to the next node in the list, a SpicyNode contains two values and a pointer to the next node. The first value is an integer carousel index, and the second value is a float spice amount. All data in this class is private and must be accessed using the public "getter" and "setter" functions within.
